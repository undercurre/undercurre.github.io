建立 Monorepo 结构

就是指在一个大的项目仓库中，管理多个模块/包（package），这种类型的项目大都在项目根目录下有一个 packages 文件夹，分多个项目管理。

通过 pnpm-workspace.yaml 实现

  1. 新建项目目录（pnpm init）

  2. 新建 packages 文件夹（组件源码）(这里是多个包)

  3. 新建 example 文件夹（组件运行的操作台）(这里也是作为一个包来做 pnpm init)

  4. 新建 pnpm-workspace.yaml

新建配置文件 .npmrc

为什么要配置 shamefully-hoist?
如果某些工具仅在根目录的 node_modules 时才有效，可以将其设置为 true 来提升那些不在根目录的 node_modules，就是将你安装的依赖包的依赖包的依赖包的…都放到同一级别（扁平化）。
