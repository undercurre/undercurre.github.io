BOM 中最核心的对象是什么？

BOM 中最核心的对象是什么？

BOM 就是 Browser object model 浏览器对象模型，它最核心的对象是 window 对象，代表了浏览器窗口本身，是所有其他 BOM 对象的顶层容器，给开发者提供了一系列的操作和管理。

浏览器窗口的功能接口，那它主要的核心对象，或者说它的有哪些关键属性呢？在这儿给大家去说一下，比如说第一个是 location，还有 Navigator，还有 document , screen, history。那么首先来说 location，它是表示当前页面的 URL 信息，可以获取或修改当前页面地址，进行页面跳转等一些操作。 Navigator 它是浏览器的信息，比如说用户代理字符串可以帮助识别浏览器类型和版本。 document 是提供当前页面文档的访问，允许操作 DOM 元素样式和事件。

screen 是用户的屏幕信息，比如说屏幕宽度高度，可用宽度和高度等等。 history 就是说记录了用户在浏览器当中的浏览历史，可以用来实现前后导航的一个功能。 innerWidth 和 innerHeight 是返回窗口内部也就是 viewport 宽度和高度，包括滚动条单步，不包括菜单栏工具栏这些浏览器的装饰。 outerWidth 和 outerHeight 是返回整个浏览器窗口的宽度和高度，包括边框菜单栏等这些浏览器装饰。OK，这个就是 BOM 的核心对象 window 以及它最常见的一些属性。
