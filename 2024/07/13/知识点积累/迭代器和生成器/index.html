迭代器和生成器

什么是迭代器？

迭代器是一种对象，它提供了一个接口，用于逐个遍历一个集合（如数组、字符串、集合等）的元素。迭代器的基本机制依赖于 next() 方法。

什么是生成器？

生成器是一个可以在函数执行过程中暂停和恢复的特殊函数。生成器函数通过 function * 语法定义，并使用 yield 关键字来暂停和恢复执行。

生成器和迭代器 next 到尽头后会发生什么？

再 next 会返回 undefined，done 是 true 表示结束

如何把一个对象解构给一个数组?

正常情况下这是不成立的，会报错：非可迭代。也就是说要解构成数组一定要是可迭代对象。

什么是可迭代对象？

满足可迭代协议的对象

什么是可迭代协议？

只要这个对象的 Symbol.iterator 属性是一个能返回迭代器或者生成器的函数

数组解构的本质是什么？

调用可迭代对象的迭代器，使用迭代器对解构变量进行赋值。

ES6 的符号是什么？

符号（Symbol） 是 ES6 引入的一种新的原始数据类型。每个从 Symbol() 创建的符号都是唯一的，并且可以作为对象属性的键，以避免属性名冲突。符号主要用于定义独特的标识符，以确保对象属性的唯一性。

ES6 知名符号是什么？

就像 Symbol.iterator,还有其他的，是一些内置的符号，它们预定义在 JavaScript 中，主要用于修改对象的行为或在内置对象中使用。
