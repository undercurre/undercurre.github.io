mysql快门式安装

手把手带小白安装mysql数据库

  1. 首先登陆这个官网链接
  2. 找到社区服务MySQL Community Server进去
  3. 下载zip压缩包
  4. 解压到你需要的地方
  5. 然后在环境变量里加入解压文件夹里的bin目录
  6. 初始化mysql：在解压文件夹里加入my.ini(记得修改其中的路径),并执行mysqld –initialize-insecure
  7. 安装mysql服务
  8. 启动Mysql服务net start MySQL8
  9. mysql -u root -p 初次登录没有密码回车就行
  10. use mysql
  11. 改密码Alter user root@localhost identified by ‘123456’;
