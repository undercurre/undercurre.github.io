那些年用的Vue-Vite插件

那些我们用过的Vue-Vite插件

必备类

@vitejs/plugin-vue

用途： 允许 Vite.js 项目中创建 Vue 组件，并通过模板<template></template>和 JavaScript 访问 Vue 组件实例。

refTransform: true 开启ref转换离开.value

把

变成

@vitejs/plugin-vue-jsx

安装提示：可能会提示没有类型声明，执行npm i @types/node -D后可解决

用途：允许你使用JSX（一种JavaScript扩展语法，JavaScript+XML，在JavaScript中使用标签语法）

允许你使用

代替

this.$createElement('div', {}, [...])

unplugin-auto-import

让你使用vue的那些需要import的东西时不用import就能使用

vite-plugin-style-import

可以自动按需引入组件库的组件样式

unplugin-vue-components/vite

自动引入组件

vite-plugin-vue-setup-extend

Vue3的setup语法糖是个好东西，但使用setup语法带来的第一个问题就是无法自定义name，解决这个问题通常是通过写两个script标签来解决，一个使用setup，一个不使用，但这样必然是不够优雅的。

vite-plugin-svg-icons

用于生成 svg 雪碧图.

在 src/main.ts 内引入注册脚本

/src/components/SvgIcon.vue

vite-plugin-html(模版必备)

一个针对 index.html，提供压缩和基于 ejs 模板功能的 vite 插件。

通过搭配 .env 文件，可以在开发或构建项目时，对 index.html 注入动态数据，例如替换网站标题。

vite-plugin-compression

使用 gzip 或者 brotli 来压缩代码。

配置说明

参数 类型 默认值 说明
verbose boolean true 是否在控制台输出压缩结果
filter RegExp or (file: string) => boolean DefaultFilter 指定哪些资源不压缩
disable boolean false 是否禁用
threshold number - 体积大于 threshold 才会被压缩,单位 b
algorithm string gzip 压缩算法,可选 [ ‘gzip’ , ‘brotliCompress’ ,’deflate’ , ‘deflateRaw’]
ext string .gz 生成的压缩包后缀
compressionOptions object - 对应的压缩算法的参数
deleteOriginFile boolean - 压缩后是否删除源文件

rollup-plugin-visualizer

打包分析

vite plugin-legacy

兼容低版本浏览器
https://zhuanlan.zhihu.com/p/583090553

vite-plugin-mock

用于模拟后端的服务

vue-router

路由管理

pinia

全局状态管理
