:before 和 ::before

:before 和 ::before 都是伪元素（pseudo-elements），用于在元素的内容之前插入一个虚拟的元素。它们的作用是让开发者可以通过 CSS 在元素的内容前插入额外的样式和内容，而不需要在 HTML 中添加额外的元素。

在 CSS2 中，伪元素使用单冒号（:）表示，例如 :before。而在 CSS3 中，为了区分伪类（pseudo-class）和伪元素，伪元素使用双冒号（::）表示，例如 ::before。不过，由于历史原因和兼容性问题，使用单冒号的写法依然被广泛使用，也被 CSS 规范所允许。

因此，无论是 :before 还是 ::before 都可以用来表示伪元素，但在新的项目中建议使用 ::before。
