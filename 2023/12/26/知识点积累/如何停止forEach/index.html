如何停止forEach

如何停止 forEach 循环

提问

你能停止 JavaScript 中的 forEach 循环吗？

提前告诉你答案

正常情况下我不建议在需要停止的时候仍然使用 forEach，但是，你要硬停，我也能干得出来！

为啥不能软停？

当我们使用“forEach”迭代数组时，回调将为数组的每个元素执行一次，并且我们无法过早地摆脱它。

怎么硬停？

1. 抛出错误

2. 设置数组长度为 0

3.使用 splice 删除数组的元素

最好是改用 some 或者 for
