纪念日统计

纪念日统计

需求

记录各种节日，并在通过微信定时提醒

模型设计

id: 一个自增的主键，用于唯一标识每个纪念日。
name: 纪念日的名称。
date: 纪念日的日期。
type: 纪念日的类型，例如生日、结婚纪念日等。
notes: 一些额外的备注信息。
reminder_days：提前提醒的天数，默认为 0 表示当天提醒。
created_at: 记录的创建时间。
updated_at: 记录的最后更新时间。

sql 语句

使用 uuid
