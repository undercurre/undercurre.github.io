嵌入式

安装 Arduino IDE：从 Arduino 官网下载并安装 IDE，这是最适合初学者的开发环境。

一. 安装 ESP32-S3 开发板支持

  1. 打开 Arduino IDE
  2. 点击 File → Preferences
  3. 在 Additional Board Manager URLs 字段中添加：https://raw.gitcode.com/qq_28547833/testall/raw/main/package_esp32_dev_index_cn.json
  4. 点击 OK 保存
  5. 点击 Tools → Board → Boards Manager…
  6. 搜索 “esp32”
  7. 找到 “ESP32 by Espressif Systems”，点击 Install

二. 安装必要的库

安装 PubSubClient 库：

  1. 点击 Sketch → Include Library → Manage Libraries…
  2. 搜索 “PubSubClient”
  3. 找到 “PubSubClient by Nick O’Leary”，点击 Install

安装 WiFi 库（通常已内置）：

ESP32 开发板包已经包含了 WiFi 库，无需额外安装

三. 配置开发板设置

  1. 选择正确的开发板：

点击 Tools → Board → ESP32 Arduino

选择 “ESP32S3 Dev Module”

  1. 重要配置：

Flash Mode: QIO

Flash Size: 16MB (128Mb)

Partition Scheme: Default 16MB with spiffs…

CPU Frequency: 240MHz (WiFi)

USB CDC On Boot: Enabled（重要！这样可以通过 USB 看到串口输出）

Upload Speed: 921600

SPIFFS 和 FATFS 的区别
功能 SPIFFS FATFS
文件系统 基于闪存的文件系统 基于磁盘的文件系统
存储大小 16MB（128MB） 依赖于磁盘大小
读写速度 快 慢
内存占用 低 高
适用场景 存储配置文件、日志等 存储大量数据文件
ESP SR 16M （3MB APP/7MB SPIFFS/2.9MB MODEL）中的 MODEL 是什么

MODEL 分区是专门用来存储语音识别模型文件的专用存储区域。

四.创建新文件（File → New）

在 Arduino IDE 中，不需要手动将文件保存为.cpp 后缀。Arduino IDE 会自动处理文件类型。

五. 上传代码到 ESP32-S3

步骤：

选择端口：点击 Tools → Port，选择你的 ESP32-S3 对应的端口（在 Windows 上通常是 COMx，在 Linux/Mac 上是/dev/ttyUSBx）

验证代码：点击 Sketch → Verify/Compile（或按 Ctrl+R）检查代码是否有错误

上传代码：点击 Sketch → Upload（或按 Ctrl+U）

如果上传失败：

确保选择了正确的开发板和端口

尝试按住 ESP32-S3 上的 BOOT 按钮，然后点击上传，当开始上传时释放按钮

检查 USB 数据线是否正常

六. 查看串口输出

点击 Tools → Serial Monitor（或按 Ctrl+Shift+M）

设置波特率为 115200

你应该看到类似这样的输出：
Connecting to YOUR_WIFI_SSID
….
WiFi connected
IP address: 192.168.1.100
Attempting MQTT connection…connected
Subscribed to: device/esp32s3/control
Message published: Hello from ESP32-S3! Time: 10000
